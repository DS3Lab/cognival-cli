Significance plots

The bars correspond to aggregated MSEs embeddings (colored) and random baselines (grey, striped), averaged across all cognitive sources of the respective modality. The MSE value for each embedding is additionally shown above each bar. In parentheses, the ratio of significant individual hypotheses (either sources, features or subjects) with respect to the random baseline is given.
